NGN Web Components

  1. ngn-cycle: For wizards, tab navigation, scrollers, or anything else that needs to display one thing at a time.
  2. ngn-layout: A quick way to create a flexbox-based page layout. Great for apps.
  3. ngn-list: A powerful and simple list management component.
  4. ngn-overlay: Creates an overlay, like the mask behind a modal window or a loading window screen.
